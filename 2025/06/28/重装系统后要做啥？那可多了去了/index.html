重装系统后要做啥？那可多了去了

发表于 2025-06-28

重装系统太多次，也许我可以把重装后要做的事情列出来，下次按列表完成。

windows篇

更新到最新后禁用自动更新！

我并不是讨厌windows自带的更新。我其实很需要windows自带的更新，它可以帮我完成一切，驱动啊补丁啊啥啥啥的，还挺实用的。我讨厌的是windows在我不想要更新的时候来更新。

所以，对不住了！

然后就可以在设置 -> 更新里面，选择推迟暂停更新5000多周

禁用位置信息，修改默认位置

对adobe 家的产品有奇效

关闭Windows defender，激活

可以临时在设置中关闭实时保护。但是过一段时间后会自动开启。

so，排除你的根目录，不让windows defender扫描才是正解。设置 -> windows安全中心 -> 病毒和威胁保护 -> 管理设置 -> 添加或删除排除项。好耶！
